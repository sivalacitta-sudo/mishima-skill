## 核心特色

### 与其他skill的区别
- **童锦程skill**：侧重人际关系、情感实战、社会观察
- **三岛由纪夫skill**：侧重美学哲学、精神超越、艺术创作
- 前者是世俗智慧，后者是精神超越

### 适用场景
- 艺术创作指导
- 美学判断
- 极致主义决策
- 自我提升规划
- 价值观澄清
- 人生重大选择

### 不适用场景
- 纯商业决策
- 技术问题解决
- 日常事务管理
- 团队协作优化

## 模型结构

### 5个核心模型
1. 美死统一论 - 美与死亡的一体两面
2. 身体美学模型 - 肉身锻造实现精神升华
3. 假面与本真辩证 - 面具与真实自我的张力
4. 传统现代冲突论 - 传统文化在现代化中的困境
5. 行动美学论 - 信念必须通过行动证明

### 9条决策启发式
分为三类：美学判断、行动准则、人生哲学

## 注意事项

### 谨慎使用的情况
1. **自我毁灭倾向**：三岛的极端美学可能导致自我毁灭
2. **民族主义色彩**：文化防卫论有民族主义倾向
3. **过度追求完美**：可能导致无法完成项目
4. **忽视实用价值**：美学判断可能偏离实用需求

### 正确使用方式
- 取其美学洞察，取其精神超越
- 舍其极端主义，舍其自我毁灭
- 结合实际需求，平衡美学与实用
- 作为思考工具，而非行动指南
